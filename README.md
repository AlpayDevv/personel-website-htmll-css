# My Personal Website

A fully responsive personal portfolio website built with **HTML and CSS**. It showcases my projects, skills, and contact information, featuring modern design elements, hover animations, and a clean layout.  

## Features

- Responsive layout for desktop and mobile
- Project cards with hover effects
- Smooth section animations using only CSS
- Modern color scheme and typography
- Navigation bar with hover effects

## Technologies Used

- HTML5
- CSS3
- CSS Grid & Flexbox
- CSS Animations & Transitions
